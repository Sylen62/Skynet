# Skynet

* Download https://nodejs.org/en/download/
* Install expo-cli
  ```
  $ npm install -g expo-cli
  ```
* Install project stuff
  ```
  $ npm install
  ```
* Run
  ```
  $ npm start
  ```
* After loads connect device to pc. Turn on wifi on device. Type a in terminal. (a - to start on android.)
  ```
  $ a
  ```
